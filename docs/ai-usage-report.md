# AI Usage Report

## Tools Used & Use Cases

### ChatGPT
I used ChatGPT during Assignment 4 for:
- Planning the final structure of the personal web application.
- Improving the portfolio from Assignment 3 into a more polished final version.
- Suggesting creative features such as a command palette, project details modal, and interactive skills tracker.
- Reviewing JavaScript logic for filtering, sorting, state management, API loading, and validation.
- Helping organize the README, technical documentation, and presentation script.

## Benefits & Challenges

### Benefits
- AI helped me organize the final project requirements into a clear checklist.
- AI gave me ideas for improving the user experience and making the site look more professional.
- AI helped identify possible bugs, such as missing IDs, weak validation, and broken links.
- AI made documentation faster by suggesting clear sections and testing steps.

### Challenges
- Some AI suggestions needed to be simplified so I could understand and explain them.
- I had to check the code manually to make sure it matched the assignment requirements.
- I needed to customize the text, project details, and design so the final work was my own.
- I had to test the website myself because AI cannot fully replace browser testing.

## Learning Outcomes
Through the use of AI, I learned:
- How to break a larger web application into smaller features.
- How JavaScript state can be stored and reused with `localStorage`.
- How to create reusable rendering functions for project and skill cards.
- How to handle API success and error states in a user-friendly way.
- How to write better technical documentation and presentation notes.
- How to use AI as a support tool while still reviewing, testing, and understanding the code.

## Responsible Use & Modifications
I used AI responsibly by:
- Reviewing AI-generated suggestions line by line.
- Modifying the code and content to fit my own assignment and portfolio.
- Testing the website in the browser after changes.
- Removing or adjusting suggestions that were unnecessary or too complex.
- Making sure I understand the JavaScript features before submitting.
- Documenting AI use clearly for academic integrity.
